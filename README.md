## このブランチについて

このブランチは、LocalStack を用いて DynamoDB をローカル環境で再現し、
CDK を使ってデプロイする構成が成立するかを検証するためのものです。

検証の結果、DynamoDB については動作可能であることが確認できましたが、

- docker-compose の構成が大きく複雑化する
- 固定 IP / network / DNS 設定が必要になる
- アプリケーション側に LocalStack 向けの分岐が発生する
- Cognito は有料版が必要で、構成の一貫性が崩れる

といった理由から、本リポジトリが重視している
「手順の単純さ」「環境差分の最小化」「再現性」の観点では
メリットが小さいと判断し、`main` ブランチには採用していません。
